================================================================================
|                                                                              |
|   Transcriptor V1.0 - Edición Élite (Guía Profesional)                       |
|                                                                              |
================================================================================

Esta aplicación automatiza la transcripción de audios a formato Word (.docx)
mediante un motor blindado de inteligencia artificial, optimizado para 
entornos forenses e institucionales en Bolivia.


================================================================================
>> INSTALACIÓN PROFESIONAL <<
================================================================================

1.  INSTALADOR
    Ejecute el archivo "Transcriptor_Setup_V1.exe". 
    El programa se instalará en: C:\Program Files\Transcriptor

2.  MODELOS DE IA (MODO OFFLINE)
    Una vez instalado, copie la carpeta "models_cache" desde su fuente de
    respaldo hacia el directorio de instalación mencionado arriba.
    * Requiere permisos de administrador.

3.  ACCESO DIRECTO
    El instalador creará un acceso directo llamado "Transcriptor" en su 
    escritorio. Úselo para iniciar el sistema de forma segura.


================================================================================
>> SEGURIDAD Y BLINDAJE <<
================================================================================

*   CÓDIGO PROTEGIDO: La lógica del sistema está compilada en archivos 
    binarios (.pyd), garantizando la integridad de la propiedad intelectual.
*   FIRMA DIGITAL: Tanto el ejecutable como el instalador están firmados 
    digitalmente por Walter Pablo Tellez Ayala.
*   SILENCIO TOTAL: El sistema inicia de forma profesional sin ventanas de 
    consola emergentes o parpadeantes.


================================================================================
>> INSTRUCCIONES DE USO <<
================================================================================

*   SELECCIÓN DE AUDIOS (Drag & Drop)
    Arrastre una carpeta o archivo individual a la interfaz.

*   MODELO Y GÉNERO
    - Modelo: Se recomienda "large-v3" para máxima precisión.
    - Profesional: Seleccione "Psicólogo" o "Psicóloga".

*   PROCESAMIENTO
    Haga clic en "Transcribir". El sistema utilizará la tarjeta NVIDIA CUDA
    para procesar los audios en segundo plano.


================================================================================
>> REQUISITOS TÉCNICOS <<
================================================================================

1.  MODO OFFLINE: 100% Funcional sin internet tras copiar los modelos.
2.  HARDWARE: Optimizado para tarjetas de video NVIDIA.


================================================================================
>> RESULTADOS <<
================================================================================

Los documentos (.docx) se guardan en la misma carpeta de los audios con el 
prefijo "ENTREVISTA INFORMATIVA_".
