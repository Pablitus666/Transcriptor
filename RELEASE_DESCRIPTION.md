# 🚀 Transcriptor — Notas de Versión

## 💎 Versión 1.1.0 — Edición Élite (Abril 2026)
Esta actualización consolida el sistema como una herramienta de grado institucional lista para despliegue masivo:

### 📦 Mejoras en la Distribución
- **Nuevo Instalador Unificado:** El archivo `Transcriptor_Setup.exe` ahora integra el entorno virtual `whisper_env` completo, eliminando errores de rutas y simplificando el proceso para el usuario.
- **Instalación en Directorio Raíz:** El programa se instala por defecto en `C:\Transcriptor`, garantizando permisos totales para logs, base de datos y archivos temporales sin intervención del usuario.
- **Portabilidad Certificada:** El software ha sido validado para funcionar desde discos externos y diferentes unidades (I:, D:, etc.) sin pérdida de configuración.

### 🖋️ Seguridad y Validación
- **Firma Digital SHA256:** El instalador y el binario principal están firmados digitalmente por **Walter Pablo Téllez Ayala**, proporcionando identidad verificada y seguridad contra manipulaciones.
- **Blindaje Binario:** Módulos centrales protegidos mediante compilación `.pyd`.

### 🛠️ Correcciones y Ajustes
- **UTF-8 Consolidado:** Se corrigieron errores de codificación en las etiquetas de la interfaz (Víctima, Configuración).
- **Estabilidad GPU:** Implementación de sincronización de hilos CUDA para evitar bloqueos durante la fase de alineación fonética.
- **Iconografía HD:** Icono multi-capa (16-256px) para una presencia visual impecable en la barra de tareas y escritorio.
